## Timeframe

| Attribute | Type | Classifier | Description                                 | Example      |
|-----------|------|------------|---------------------------------------------|--------------|
| from      | date | required   | The from-date of the timeframe (inclusive). | `2017-02-01` |
| to        | date | required   | The to-date of the timeframe (inclusive).   | `2017-03-01` |
